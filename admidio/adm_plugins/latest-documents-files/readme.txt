 1. Introduction
-------------------------------------------------------------------------
This plugin will list the last uploaded files with a link to download
them.

Sidebar Downloads is a plugin of the online membership management
Admidio (https://www.admidio.org) and can not be used without this.


 2. Installation
-------------------------------------------------------------------------
Please take a look at our Wiki for a detailed description how to integrate
a plugin to your Admidio installation:
https://www.admidio.org/dokuwiki/doku.php?id=en:plugins:installation


 3. Update
-------------------------------------------------------------------------
To update this plugin just replace the whole folder of the birthday plugin.

If there are new configuration parameters within the new version and you 
would like to use a different value for this parameters than the default you 
must add this new parameters to your existing config.php of this plugin. 
You can view the default settings in the config_sample.php


 4. Bugs
-------------------------------------------------------------------------
If you found an issue with this plugin you can report this at our
plugin forum:
https://www.admidio.org/forum/viewforum.php?f=10

It is also possible to report an issue directly in our issue tracker at
GitHub. There, however, it should be ensured that this is not a
configuration issue, but in fact an error of the plugin.
https://github.com/Admidio/admidio/issues
